# SET AI Component — UFPS

NLP fine-tuning pipeline for automated analysis of student teaching evaluations at Universidad Francisco de Paula Santander.

## What it does

- **Task 1:** Classifies student comments by risk level → `BAJO` / `MEDIO` / `ALTO`
- **Task 2:** Classifies comments by pedagogical category (multi-label) → up to 5 simultaneous labels

## Models

| Folder | Model | HuggingFace ID |
|---|---|---|
| `RoBERTuito/` | RoBERTuito | `pysentimiento/robertuito-base-uncased` |
| `BETO/` | BETO | `dccuchile/bert-base-spanish-wwm-cased` |
| `DistilBETO/` | DistilBETO | `dccuchile/distilbert-base-spanish-uncased` |
| `Electricidad/` | Electricidad | `mrm8488/electricidad-base-discriminator` |

Each folder contains:
- `entrenamientoNivelDeRiesgo.py` — Task 1 training
- `entrenamientoCategorias.py` — Task 2 training
- `predecir-<model>.py` — Interactive CLI inference (both tasks)

## Dataset

- `data/dataset_tesis_version_3_categories_new.csv` — original (Task 1)
- `data/dataset_tesis_limpio.csv` — cleaned (Task 2), generated by `correctionData.py`
- 9,457 manually labeled student comments in Spanish

## Installation

```bash
# 1. Create virtual environment
python -m venv .venv
.venv\Scripts\activate          # Windows
source .venv/bin/activate       # macOS / Linux

# 2. Install PyTorch with CUDA 12.4
pip install torch==2.6.0 torchvision torchaudio --index-url https://download.pytorch.org/whl/cu124

# 3. Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
# Clean dataset (required before Task 2 training)
python correctionData.py

# Train — example with RoBERTuito
cd RoBERTuito
python entrenamientoNivelDeRiesgo.py   # Task 1
python entrenamientoCategorias.py      # Task 2

# Run inference
python predecir-robertuito-categorias.py
```

> Trained models are excluded from this repo (~400MB each). Train locally before running inference.

## Environment
